Interjecciones

Son aquellas palabras o expresiones que no forman parte de la oración. Además, las conjunciones son palabras o conjuntos de palabras que sirven para expresar un sentimiento vivo, llamar la atención o describir un sonido.


Es decir son palabras cuya función es expresar sentimientos (¡ay!), impresiones (¡oh!), reacciones (¡ah!), actitudes de manera directa e inmediata, sin seguir una estructura gramatical establecida. Como por ejemplo:

  • ¡Alto!: se usa como llamada de pausa imperativa
  • ¡Ay!: se emplea para expresar un sentimiento vivo
  • ¡Eh!: se usa para preguntar, llamar, despreciar, reprender o advertir
  • ¡Hola!: se usa como salutación de apertura de conversación
  • ¡Ey!: se usa como expresión de apertura de llamar a una persona

Las interjecciones se consideran como oraciones independientes con significado propio. Por eso las interjecciones se deben escribir siempre entre dos signos de puntuación (dos comas, dos signos de admiración, dos signos de interrogación, el principio de la frase y una coma, etc.).

Ejemplos:

  1. Interjecciones propias: una sola palabra entre signos de exclamación o interrogación

    2. ¡AH!



  2. Interjecciones impropias: son sustantivos, verbos, adjetivos, etc. que se utilizan como interjecciones

    3. ¡Caracoles!



  3. Interjecciones imitativas: sirven para describir sonidos de onomatopeya.

    4. ¡Boom!



  4. Interjecciones Expresivas: sirve para expresar una emoción.

    5. ¡WoW!
